## 介绍 GitLens+

当您使用免费账户登录时，GitLens+ 会添加全新的、完全可选的功能，增强您当前的 GitLens 体验。免费的 GitLens+ 账户可让您在公共仓库中使用这些新的 GitLens+ 功能，而付费账户则允许您在私有仓库中使用它们。

🛈 所有其他 GitLens 功能始终可以访问，无需账户，并将继续不断发展和投入。

[了解更多关于 GitLens+ 的信息](https://gitkraken.com/gitlens/plus-features '了解更多')

首批新的 GitLens+ 功能是“可视化文件历史”和“工作区 (Worktrees)”。

## 这会影响现有功能吗？

不，GitLens+ 的引入对现有的 GitLens 功能没有影响，因此您不会失去对任何您熟悉和喜爱的 GitLens 功能的访问权限。事实上，我们正在大力投入增强和扩展 GitLens 功能集。创建一个账户只是为了让您能够访问一组新功能，从而在 VS Code 中进一步利用 Git 的力量！

## 更多 GitLens+ 功能即将推出

更多功能（如可视化提交图）以及针对 GitHub Enterprise 和 GitLab 的集成即将推出。
